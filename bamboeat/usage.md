# BambEat | Usage

## How to Eat Bamboo

Hold bamboo in your hand and right-click (use item). If you have the `bambeat.eat` permission and meet the hunger conditions set in `config.yml`, the bamboo will be consumed and hunger/effects will be applied.

---

## Commands

| Command | Permission | Description |
|---|---|---|
| `/bambeat version` | *(none)* | Shows plugin version and authors |
| `/bambeat reload` | `bambeat.admin` | Reloads `config.yml` |

---

## Permissions

| Permission | Default | Description |
|---|---|---|
| `bambeat.eat` | `true` | Allows the player to eat bamboo |
| `bambeat.admin` | OP | Access to admin commands |
