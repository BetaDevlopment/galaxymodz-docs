# SpawnControl | Overview

**SpawnControl** (also known as **OrbSpawn**) is a GalaxyModz plugin that gives server administrators full control over entity and orb spawning behaviour | per entity type, per region, and per world.

---

## What It Does

SpawnControl lets you define, limit, schedule, and customize how entities and orbs spawn. Whether you want to cap spawn rates, trigger spawns on events, or create entirely custom spawn profiles, SpawnControl handles it without replacing Minecraft's native spawn system.

**Key Features:**

- Fine-grained spawn rate control per entity type
- Event-triggered spawning
- Region-based spawn zones
- Full `config.yml` and `spawns.yml` configuration
- Developer API for plugin integration

---

## Use Cases

- Prevent mob overcrowding on survival servers
- Create custom orb drop systems for minigames
- Schedule timed spawns for events or boss encounters

---
